1 软件下载

（1）下载安装node.js，一般直接下一步下一步安装

（2）下载安装git，一般直接下一步下一步安装

win+R：cmd，打开windows终端

（3）下载安装Hexo

2 github新建仓库

在github中新建仓库，new repository

  • Repository name：fengzhenchn.github.io
  • 勾选：Add a README file
  • Create Repository

3 部署本地Git与Github连接

（1）生成SSH KEY

随便进入电脑中的任意文件夹，右键，git bash here

（2）复制.pub文件内容

在电脑路径C:\Users\FZ\.ssh中，用记事本打开id_rsa.pub文件，全选复制内容

（3）将KEY添加到github的SSH Key里

打开github—settings—SSH and GPG keys—New SSH key

  • title无所谓：Blog
  • key：刚才的复制的id_rsa.pub文件里内容，粘贴到这里
  • Add SSH key

git中输入下例命令，确定是否绑定成功

（4）右键打开git bash，设置用户名和电子邮箱

4 初始化本地博客文件夹

新建文件夹命名为Blog，进入Blog文件夹，右键，gti bash here

通过浏览器查看http://localhost:4000/是否成功

5 butterfly主题安装

通过npm安装的主题会放node_moduels里，然后在之前创建的Blog根目录下创建 _config.butterfly.yml，将butterfly主题下的 _config.yml内容复制进去。

修改Blog目录中的 _config.yml，修改两个字段：

更新部署博客页面

6 博客部署到github

修改Blog根目录下的_config.yml文件中的deploy

安装 hexo-deployer-git 自动部署发布工具，将 hexo 部署到 git page 的 deployer：在Blog根目录下，右键，gti bash here

现在就可以通过：fengzhenchn.github.io访问博客了
