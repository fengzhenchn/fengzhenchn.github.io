2022-12-10_论文排版

发表于2022-12-10|更新于2023-12-15|学习

F4 键的作用是重复上一步操作，在 PPT 和 Excel 中也是同样的作用。

ctrl+P：打印预览

ctrl+Y：恢复上一步操作

ctrl+Z：撤销上一步操作

ctrl+enter：分页符

按下Alt后，会出现符号，按下对应的符号就可以执行对应的功能

1 准备工作

【文件】—【选项】—【显示】—勾选“显示所有格式标记”

2 页面整体布局

【布局】—【页面设置】—【页边距】—【自定义页边距】

  • 上下左右：2.54 厘米

3 论文正文

【开始】—【样式】—【创建样式】—名称：论文正文—【修改】

  • 样式基准：正文
  • 后续段落样式：论文正文
  • 勾选：基于该模板的新建文档
  • 格式
    • 字体—中文字体：宋体，西文字体：Times New Roman，小四，两端对齐
    • 段落—左侧右侧 0 字符，首行缩进 2 字符，段前段后 0 行，行距固定值：22 磅

如果希望新样式显示在所有新的 Word 文档中，请在“样式”库中右键单击它，单击“修改”，然后在对话框底部选择“基于此模板新建文档”。

4.1 定义多级列表

【开始】—【段落】—【多级列表】—【定义新的多级列表】—【更多】（设置完后最后再确定）

    • 勾选：基于该模板的新建文档
    • 格式
      • 字体—宋体，Times New Roman，小三号，加粗
      • 段落—左侧右侧 0 字符，首行缩进 0 字符，段前段后 0 行，行距固定值：22 磅

  • 右键“标题二”样式—【修改】
    • 后续段落样式：论文正文
    • 勾选：基于该模板的新建文档
    • 格式
      • 字体—宋体，Times New Roman，四号，加粗
      • 段落—左侧右侧 0 字符，首行缩进 0 字符，段前段后 0 行，行距固定值：22 磅
    • 确定

5 表格和图片

5.1 三线表

【插入】—【表格】—【插入表格】—勾选：根据窗口调整表格，为新表格记忆此尺寸—确定

【表设计】—【表格样式】—【新建表格样式】

  • 名称：三线表
  • 勾选：基于该模板的新建文档
  • 将格式应用于：整个表格
    • 格式—边框和底纹—自定义
      • 上粗下细的线，1.5 磅，应用于上框线
      • 上细下粗的线，1.5 磅，应用于下框线
      • 确定
  • 将格式应用与：标题行
    • 找到一条细线，0.5磅，下框线
  • 确定

跨页显示表头：

选中表头行，右键—表格属性—行—勾选：在隔页顶端以标题行形式重复出现—确定

5.2 表格标题：表上

【引用】—【题注】—【插入题注】—【自动插入题注】

  • 插入时添加题注：Microsoft word表格
  • 使用标签：表
  • 位置：项目上方
  • 确定

【开始】—【样式】—【创建样式】—名称：表格标题—【修改】

  • 勾选：基于该模板的新建文档
  • 格式
    • 字体—宋体，小四号，居中
    • 段落—左侧右侧 0 字符，首行缩进 0 字符，段前段后 0 行，行距固定值：22 磅
  • 确定

5.3 图片标题：图下

【开始】—【样式】—【创建样式】—名称：图片标题—【修改】

  • 勾选：基于该模板的新建文档
  • 格式
    • 字体—宋体，五号，居中
    • 段落—左侧右侧 0 字符，首行缩进 0 字符，段前段后 0 行，行距固定值：22 磅
  • 确定

6 公式

【开始】—【样式】—【创建样式】—名称：公式编号—【修改】—勾选：基于该模板的新建文档

  • 字体：宋体，小四号，居中
  • 格式
    • 制表位
      • 制表位位置：19.5—居中对齐—引导符：1 无—设置
      • 制表位位置：34—右对齐—引导符：1 无—设置
      • 【确定】
    • 段落—中文版式—文本对齐方式：居中
  • 确定

公式居中的办法：
选中公式和编号—点击公式编号样式—将光标放在公式前，按TAB键一下—将光标放在
公式和编号之间，按TAB键两下

7 参考文献

开始—段落—编号—定义新编号格式

  • 标号格式：[1]，对齐方式：左对齐

引用—题注—交叉引用—选择好文献—插入—设置上标格式

8 标题、摘要、关键词

标题：黑体，三号，居中，加粗

“摘要”：黑体，小三号，居中，加粗

“Abstract”：Arial Black，小三号

“关键词”：黑体，小四号，加方括号

“Key words”：Arial Black，小四号，加方括号

中文摘要和关键词正文：楷体，小四号

英文摘要和关键词正文： Time New Roman，小四号

9 页码

光标放在摘要和关键词页末—布局—页面设置—分隔符—分节符—下一页

插入—页眉和页码—页码—设置页码格式—页码编号—起始页码：1

插入—页眉和页码—页码—页面底端—普通数字2

双击正文第一页页码—取消链接到前一节—双击摘要页页码—删除摘要页码—关闭页眉页脚

10 模板文件保存

将文件保存为 .dotx 格式，放入文件夹中 C:\Users\FZ\Documents\自定义 Office 模板，

【选项】—【保存】—默认个人模板位子：C:\Users\FZ\Documents\自定义 Office 模板

之后就能看到这个模板文件了。
