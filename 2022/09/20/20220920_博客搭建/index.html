2 github 新建仓库

3 部署本地 Git 与 Github 连接

（1）生成 SSH KEY：

随便进入电脑中的任意文件夹，右键，git bash here

（2）复制 .pub 文件内容：

在电脑路径 C:\Users\FZ\.ssh 中，用记事本打开 id_rsa.pub 文件，全选复制内容

（3）将 KEY 添加到 github 的 SSH Key 里

Github—settings—SSH and GPG keys—New SSH key
title 无所谓：Blog
key：刚才的复制的 id_rsa.pub 文件里内容，粘贴到这里
Add SSH key

git 中输入下例命令，确定是否绑定成功

（4）git 中输入下例命令，设置用户名和电子邮箱

4 初始化本地博客文件夹

（1）hexo 配置

新建文件夹命名为 Blog，进入 Blog 文件夹，右键，gti bash here

通过浏览器查看 http://localhost:4000/ 是否成功

（2）butterfly 主题安装

通过 npm 安装的主题会放文件夹 node_moduels 里，然后在之前创建的 Blog 文件夹根目录下创建 _config.butterfly.yml 文件，将 butterfly 主题下的 _config.yml 内容复制进去。

5 博客部署到github

修改 Blog 根目录下的 _config.yml 文件中的 deploy：

安装 hexo-deployer-git 自动部署发布工具，将 hexo 部署到 git page 的 deployer：在 Blog 根目录下，右键，gti bash here

现在就可以通过：https://fengzhenchn.github.io/ 访问博客了

6 butterfly 主题配置

6.1 页面设置

6.1.1 标签页

git 中输入 hexo new page tags

修改根目录下的 source/tags/index.md 这个文件，添加 type: "tags" 这一行

6.1.2 分类页

git 中输入 hexo new page categories

修改根目录下的 source/categories/index.md 这个文件，添加 type: "categories" 这一行

6.1.3 友情链接（未成功）

创建友情链接页面

git 中输入 hexo new page link

修改根目录下的 source/link/index.md 这个文件，添加 type: "link" 这一行

添加友情链接-本地生成友情链接
